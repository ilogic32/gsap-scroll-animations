================================================================================
UPLOAD VIA FTP/SFTP
================================================================================

1. Connect to your WordPress hosting via FTP
2. Navigate to: /wp-content/plugins/
3. Upload the entire "GSAP-Animations" folder
4. Verify folder structure is preserved:
   - GSAP-Animations/
   - GSAP-Animations/gsap-scroll-animations.php
   - GSAP-Animations/assets/css/gsap-scrolltrigger.css
   - GSAP-Animations/*.md files

================================================================================
UPLOAD VIA WORDPRESS ADMIN
================================================================================

1. Compress the GSAP-Animations folder as: GSAP-Animations.zip
2. Go to WordPress Admin → Plugins → Add New
3. Click "Upload Plugin"
4. Select GSAP-Animations.zip
5. Click "Install Now"
6. Click "Activate Plugin"

================================================================================

QUICK START (After Upload)
================================================================================

ID-BASED (Admin UI):
1. Settings → GSAP Animations
2. Click "+ Add Row"
3. Enter HTML ID and select animation type
4. Save settings
5. Add to page: <div id="my-element">Content</div>

CLASS-BASED (HTML):
1. Settings → GSAP Animations
2. Check "Enable Class-Based Animation System"
3. Save settings
4. Add to page: <div class="gsap-fade-up">Content</div>
