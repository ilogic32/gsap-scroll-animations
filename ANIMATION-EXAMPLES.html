üéØ Parallax Effects (3 Animations)
1. Vertical Parallax
class-based

Element moves slower than scroll speed, creating depth effect

<!-- Vertical parallax (moves upward as you scroll down) --> <div class="gsap-parallax" data-speed="-5"> <img src="background.jpg" alt="Parallax Background"> </div>
Data Attributes:
  • data-speed - Speed multiplier (default: -5)
  • Negative values move backward, positive move forward

üí° Tips & Best Practices
‚úÖ Use Meaningful IDs for ID-Based:

Use descriptive IDs like hero-section, testimonial-box instead of item1, item2

‚úÖ Combine Multiple Classes:

You can combine a class-based animation with gsap-mobile-disable to disable on mobile: <div class="gsap-fade-up gsap-mobile-disable">

‚úÖ Use Data Attributes for Customization:

Always use data attributes to customize: data-duration, data-delay, data-distance

‚ö†Ô∏è Avoid Excessive Animations:

Don't animate everything! Too many animations can distract and harm performance. Use animations to highlight important content.

‚ö†Ô∏è Test on Mobile:

Always test your animations on actual mobile devices or use DevTools emulation. Some effects may not look good on small screens.

‚ö†Ô∏è Keep Accessibility in Mind:

The plugin automatically respects prefers-reduced-motion, so users with motion sensitivity won't see animations.
